### Measure Culture Change

Keeping track of the program is the top priority to promote culture change and adjust the program based on our metrics. We can use corporate packaged tools like Microsoft 365 (PowerBI, Excel), Tableau, Google Sheets. 

Training Evaluation Framework: [Kirkpatrick Evaluation ](https://www.mindtools.com/ak1yhhs/kirkpatricks-four-level-training-evaluation-model)

What you can measure
- Compliance Trainings and Policy Signoff
- Phishing Simulation Training Reports
- User's Compliance Violations
- User's Micro-Training Completions
- Real Phishing Clicks

How you can provide risk reduction metrics on FAIR standards 
- Value assessments 
- Risk assessments (Reduction in Annualized Loss Expectancy(ALE))
- https://www.risklens.com/
